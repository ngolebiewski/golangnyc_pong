# Code Pong in Go with Ebitengine Tutorial/A live coding talk by Nick Golebiewski.

![Gopher as a ball in pong, video game animated gif](/gopher-pong.gif)

Pong is the best known early video game! So lets make that with Go using the open source Ebitengine Game Engine developed by Hajime Hoshi.

Why? Go is versatile!

Although Go is primarily known as a backend language, used for DevOps (Development Operations), REST APIs, Cloud Computing, Network Engineering, Site Reliability Engineering etc. out in the real digital world. It is also fast and compiles down to a binary. 

Every ebitengine project starts here:
https://ebitengine.org/en/documents/install.html 


**Play online: https://gopher-pong.netlify.app/**

Controls:
* P1:W/S  P2:arrows - up/down
* r - reset
* f - fullscreen
* space - serve ball
* g - gopher mode (online only, didn't make it into our talk, or this code version, but is online)
